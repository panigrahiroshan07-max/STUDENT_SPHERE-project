# 🎓 Student Sphere

Student Sphere is a student productivity and academic management web application developed to help students organize their academic life in one place. The platform combines study planning, attendance tracking, exam scheduling, assignment management, goal setting, progress monitoring, and smart reminders to improve productivity and learning efficiency.

This project was created for the **ByteBattle Hackathon** under the **Education & Student Life Innovation** theme, focusing on solving real challenges faced by students through technology.

---

## 🚀 Problem Statement

Students often use multiple apps and notebooks to manage their studies, attendance, assignments, exams, and goals. This leads to poor organization, missed deadlines, and increased academic stress.

Student Sphere addresses this problem by providing a centralized dashboard where students can manage all their academic activities efficiently.

---

## ✨ Features

### 📚 Subject Management

* Add and manage subjects.
* Delete subjects when no longer required.
* Automatically updates dashboard statistics.

### 📅 Weekly Timetable

* Create a personalized study timetable.
* Assign subjects to specific days and times.
* Edit and delete timetable entries.

### 📊 Attendance Tracker

* Mark lectures as Present or Absent.
* Track subject-wise attendance percentage.
* View overall attendance percentage.
* Attendance alerts for subjects below 75%.

### 🎓 Exam Schedule

* Add upcoming exams with dates.
* Automatic countdown showing days remaining.
* Exam reminders displayed in the Alerts section.

### 📝 Assignment Manager

* Add assignments.
* Mark assignments as completed.
* Edit or delete assignments.
* Pending assignments appear in reminders.

### 🎯 Study Goals

* Create daily or weekly study goals.
* Track completion status.
* Manage and update goals easily.

### 📈 Progress Tracker

* Calculates completion percentage based on assignments and goals.
* Visual progress bar for motivation.
* Dashboard progress statistics.

### 🔔 Smart Alerts & Reminders

* Upcoming exam alerts.
* Pending assignment reminders.
* Pending study goal reminders.
* Attendance warning notifications.

### 💾 Local Storage Support

* Data is automatically saved in browser local storage.
* Information remains available even after refreshing the page.
* No account creation required.

### 📱 Responsive Design

* Mobile-friendly interface.
* Sidebar navigation menu.
* Clean and modern user experience.
